# rajcinapad
rajcinapad is a six key macropad with a rotary encoder and lights
it is a project i made for hackclub blueprint

# Features
- It has a rotary encoder switch that can change volume 
 i also use it to change the leds
- the pcb looks cool

# BOM
- 6x Cherry MX Switches
- 6x DSA Keycaps
- 1x EC11 Rotary encoder switch
- 4x SK6812 MINI neopixels
- 1x XIAO RP2040
- 1x Case (2 parts)
- 4x M3x16mm screws
